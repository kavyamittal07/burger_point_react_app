# 🍔 Burger Point React Application

A modern and interactive burger builder web app built using React. Users can customize their burger by adding or removing ingredients and see the total price update in real time.

---

## 🚀 Features

- 🍔 Build your own burger
- ➕ Add / Remove ingredients
- 💰 Dynamic price calculation
- 🛒 Order summary modal
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 CSS
- 📦 npm
